# 🛒 MOKOLO Market — Plateforme E-Commerce B2B & B2C (Modèle Alibaba)

Propulsé et développé par **AbJ Tech**. 
**MOKOLO Market** est une marketplace multi-vendeurs innovante et ultra-sécurisée adaptée au marché africain, avec une implémentation initiale pour le Cameroun.

## 🛡️ Fonctionnalités Clés du MVP

* **Architecture Hybride NoSQL** : Collection unique et centralisée pour un catalogue global unifié.
* **Authentification Avancée** : Double rôle Acheteur / Vendeur cloisonné avec vérification de session.
* **Système de Séquestre Garanti** : Les fonds collectés via les paiements mobiles sont bloqués sur le compte central et ne sont libérés au vendeur qu'après validation de la livraison par l'acheteur via une note 5 étoiles.
* **Paiements Mobiles Locaux** : Intégration transparente pour MTN MoMo et Orange Money.
* **Automatisation CRM (Resend)** : Relances et notifications automatisées (limitation stricte à 3 emails max par vendeur).
* **Console Logistique Temps Réel** : Tableau de bord réactif (`onSnapshot`) pour la prise en charge instantanée des colis par les livreurs.

## 🛠️ Stack Technique

* **Framework Front-end** : Next.js 14 (App Router) & TypeScript
* **Design System** : Tailwind CSS & Lucide Icons (Charte Rouge `#D72638`, Noir, Blanc)
* **Base de données & Auth** : Firebase Client SDK (Firestore & Auth)
* **Backend Transactionnel** : Cloud Functions (Firebase Admin SDK)
* **Routage Notification** : Resend API
